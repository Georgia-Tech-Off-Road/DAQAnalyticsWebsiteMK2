# Overview
Backend folder for our data analytics website

# Quickstart
Note, this quickstart  assumes that Node.js, NPM, and Python are already installed.

#### Install Libraries
``npm install``

``pip install flask``

#### Create Database, Populate with Dummy Values
``npm run init-dev``
If you get an error after running the previous command, then you likely already have database at database/data.db.

#### Start Server
``npm run start``

Backend should start on localhost:3000, microservices localhost:5000.
